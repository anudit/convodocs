# TrustScore

## Introduction

TrustScore aggregates the multiple on-chain social graphs acting as *sources of reputation* that count towards some points in building the overall TrustScore.

Current Integrations
- Proof of Humanity
- BrightId
- POAP
- ENS
- Idena
- CryptoScam DB
- Unstoppable Domains
- Uniswap Sybil
- DeepDAO
- Boardroom
- RabbitHole
- Mirror
- SuperRare
- Rarible
- Foundation
- AsyncArt
- KnownOrigin
- Coinvise

## Integration

```html
https://theconvo.space/api/identity?address=<ethereum-address>&apikey=CONVO
```
