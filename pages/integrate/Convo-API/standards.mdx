# Standards

## Introduction

Here we document the cross-platform standards for setting up your url & threadid schema for storing and querying conversational data.

### NFTs

- URL: Your Website Address like [https://huddln.io/](https://huddln.io/)
- ThreadId: `<tokenContractAddress>:<tokenId>:<chainId>`
