# Omnid RPC

## Introduction
Omnid's RPC is an [open-source](https://github.com/anudit/omnid-proxy-rpc), thin, self-hostable RPC Proxy that scans the transactions for malicious/phishing addresses using static analysis and transaction simulation before you send them and reverts if a match is found before sending the transaction.
