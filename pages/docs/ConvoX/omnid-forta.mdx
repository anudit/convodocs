### RPC API

You can query the Alerts on any arbitary address using Metamask's RPC API, [view the docs](/docs/ConvoX/omnid-metamask#omnid_getfortadata).

You can query the Identity data for any arbitary address using Omnid's API, [view the docs](/docs/Convo-API/omnid#integration).
