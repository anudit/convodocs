# SDK - Identity

## Get TrustScore for Address

```javascript
convo.identity.getTrustScore(
    signerAddress,  // Ethereum Address
)
```

Example,
```javascript
let resp = await convo.identity.getTrustScore(
    "0x707aC3937A9B31C225D8C240F5917Be97cab9F20"
);
```
