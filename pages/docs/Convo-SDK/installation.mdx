## Setup

1. Generate and Copy your API key from [https://theconvo.space/dashboard/developer/](https://theconvo.space/dashboard/developer)
2. Import Convo Space in your own project.

    <Tabs items={['ESM', 'CJS', 'UMD', 'ESM.run']} defaultIndex={0}>
    <Tab>

    ```javascript
    import { Convo } from '@theconvospace/sdk';
    ```

    </Tab>
    <Tab>

    ```javascript
    const { Convo } = require('@theconvospace/sdk');
    ```

    </Tab>
    <Tab>

    ```javascript
    const { Convo } = convo;
    ```

    </Tab>
    <Tab>

    ```javascript
    import { Convo } from 'https://esm.run/@theconvospace/sdk';
    ```

    </Tab>
    </Tabs>

3. Setup an new instance of Convo using

    ```javascript
    const convo = new Convo("apikey")
    ```

    Replace `apikey` with your newly generated key.
